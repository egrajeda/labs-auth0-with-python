# README

Playing around with Auth0 and Python.

![Demo](https://cloud.githubusercontent.com/assets/12800/25068176/776c47de-2231-11e7-9950-114f21adc2a3.gif)

To run it use:

```
$ FLASK_DEBUG="1" \
    AUTH0_HOST="USERNAME.auth0.com" \
    AUTH0_CLIENT_ID="..." \
    AUTH0_CLIENT_SECRET="..." \
    python src/application.py
```
